API Reference

Complete documentation for the NeuroWeb API

REST API

GET /api/v1/models

List all available AI models

curl -X GET "https://api.neuroweb.ai/v1/models" \
-H "Authorization: Bearer YOUR_API_KEY"
POST /api/v1/generate

Generate text using specified model

curl -X POST "https://api.neuroweb.ai/v1/generate" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
    "model": "gpt2",
    "prompt": "Once upon a time",
    "max_length": 100
}'

WebSocket API

Real-time communication for streaming responses

const socket = new WebSocket('wss://api.neuroweb.ai/v1/ws');
socket.onmessage = (event) => {
    const response = JSON.parse(event.data);
    console.log(response);
};

Authentication

Secure your API requests using API keys

API Key Generation

  1. Log in to your NeuroWeb account
  2. Navigate to API Settings
  3. Click "Generate New Key"
